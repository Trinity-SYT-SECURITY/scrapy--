根據主網頁所提供的超連結，程式會進入該連結做爬蟲的動作，並將結果導出成txt檔案並另外建立目錄存取，若是爬下來的都是英文內容，可以用我另外一個程式把txt檔翻譯成繁體中文，可惜的是，他有翻譯字數限制，因為是免費的..

**程式不能直接執行，請根據scrapy的手冊先建立執行環境，在將要執行的檔案丟到spider目錄下**

程式內的所有css定位元素都要自行更改，不然每個網站都不會相同的適用
